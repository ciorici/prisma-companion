== Description ==

Prisma Companion is an optional companion plugin for [Prisma Core](https://github.com/wpzoom/prisma-core) theme. It adds additional features such as widgets, blocks and a collection of pre-built websites (demos) for popular page builders and the WordPress block editor (Gutenberg).

== Installation ==

= Automatic Installation =

Automatic installation is the easiest way to install Prisma Companion as WordPress handles the file transfers itself. To do an automatic install of Prisma Companion, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Prisma Companion" and click Search Plugins. Once youâ€™ve found our plugin simply click "Install Now". After the installation has finished, click the "Activate Plugin" button.

= Manual Installation =

1. Upload the "prisma-companion.zip" folder to your "/wp-content/plugins" directory or alternatively upload the "prisma-companion.zip" file via the "Add New" button in WordPress plugins page.
2. Activate Prisma Companion.
